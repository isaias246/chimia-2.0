# CHEMIA

## Plataforma Educativa de Química en Español

CHEMIA es una plataforma de aprendizaje de química diseñada para estudiantes de nivel secundario, universitario e ingeniería.

Su objetivo es combinar las mejores características de:

- Photomath
- Quimify
- Khan Academy
- Wolfram Alpha

en una única plataforma especializada en química.

---
